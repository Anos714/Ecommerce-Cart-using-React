# üõí React Shopping Cart Application

A fully responsive E-commerce Shopping Cart application built with **React.js**. This project demonstrates modern state management using the **Context API**, data persistence with **LocalStorage**, and responsive styling with **Tailwind CSS**.

## üöÄ Features

- **Product Listing:** Fetches and displays products dynamically from an external API.
- **Shopping Cart:**
  - Add items to the cart.
  - Remove items from the cart (with instant UI updates).
  - Prevent duplicate items from being added.
- **Cart Persistence:** Uses `localStorage` so items remain in the cart even after refreshing the page.
- **Dynamic Totals:** Automatically calculates the total price of items in the cart.
- **Product Details:** Dedicated page for viewing individual product details.
- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop using Tailwind CSS.
